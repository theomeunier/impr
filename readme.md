Site L'imprimeur
----

### Access

- web : http://localhost:8181
- phpmyadmin : http://localhost/8585
- mailhog : http://localhost:8025

### Installation

- Installer les dépendances JS :
```
npm install
```

- Installer les dépendances PHP :
```
composer install
```

- Installer les fichiers CSS :
```
npm run process-styles
```
